# Main Character Plot Armor

A RimWorld mod that adds a "Main Character Plot Armor" trait, making a single pawn dramatically overpowered. Ideal for single-pawn colonies and story-focused playthroughs where you want one character who simply cannot die.

Languages: [English](#main-character-plot-armor) · [Español](#español) · [Русский](#русский) · [Français](#français) · [Deutsch](#deutsch) · [Italiano](#italiano) · [日本語](#日本語) · [한국어](#한국어) · [中文](#中文简体)

## What the trait does

A pawn with this trait gets:
- Heavy damage resistance (incoming damage reduced to 1%)
- Maximum armor against sharp, blunt, and heat (200% — the cap)
- 95% melee dodge chance
- 10x faster injury healing
- 3x immunity gain speed
- High pain shock and mental break thresholds (rarely breaks)
- Permanent +20 mood buff
- 1.5x move speed, 4x research speed, 4x learning rate
- 10x carrying capacity
- Extreme shooting accuracy at all ranges
- Near-zero mortar miss radius
- 90% reduced flammability
- Wide comfortable temperature range
- +2 beauty
- Conflicts with the Wimp and Frail traits

## Supported versions

RimWorld 1.3, 1.4, 1.5, and 1.6. No DLC required.

## Installation

**Steam Workshop:** Subscribe at https://steamcommunity.com/sharedfiles/filedetails/?id=3208213612.

**Manual:** Copy the `MainCharacterPlotArmor` folder into your RimWorld `Mods` directory.

Enable the mod in the Mods menu in-game.

## Usage

The trait has a `commonality` of `0`, so it will never appear on randomly generated pawns. To assign it:

- Use **Character Editor** or **EdB Prepare Carefully** during pawn creation, or
- Use **dev mode** (Options → Development mode) to add the trait to an existing pawn.
